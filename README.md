# Muhammad Hamza — Portfolio

Personal portfolio site for Muhammad Hamza, Full Stack AI Engineer. Built with Next.js 16, React 19, TypeScript, Tailwind CSS v4, GSAP, and Lenis.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

- `app/` — App Router pages and layouts
- `components/` — Section components (hero, work, services, etc.)
- `lib/portfolio-data.ts` — All static content (projects, experience, skills)

## Scripts

| Command            | Description              |
| ------------------ | ------------------------ |
| `npm run dev`      | Start development server |
| `npm run build`    | Production build         |
| `npm run lint`     | Run ESLint               |
| `npx tsc --noEmit` | Typecheck                |
